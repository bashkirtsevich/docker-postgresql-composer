# PostgreSQL with pgAdmin

Dockerized PostgreSQL-server with pgAdmin web-ui.

## Services handling

* Ephemeral
  * startup: `./ephemeral.sh`
  * stop: `./ephemeral-stop.sh`
* Persistent
  * startup: `./persistent.sh`
  * stop: `./persistent-stop.sh`
